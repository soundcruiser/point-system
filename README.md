
# ポイント管理システム (QR Point Admin)

施設内限定で使用することを目的とした、スタッフ向けのポイント管理・QRスキャンアプリです。
スタッフが来所した利用者のQRコードを読み込むことで出席ポイントを付与し、利用者は貯まったポイントを使って商品を交換することができます。

## 🎯 主な機能

- **出席登録（ポイント付与）:**
  - スタッフが利用者のQRコードをスキャンすると、1回につき10ポイントが付与されます。
  - ※1日1回の制限があり、同日に複数回スキャンしても重複して付与されない仕様です。
- **商品交換:**
  - 貯まったポイントを消費して、アプリ内の商品（コーヒー: 50pt、プリント: 10pt、肉まん: 100pt）と交換できます。
- **利用者管理:**
  - 新規利用者のIDと名前を登録でき、現在の保有ポイント一覧を確認できます。
- **履歴管理:**
  - 誰がいつポイントを獲得したか、または何の商品と交換したかの履歴を一覧で確認できます。
- **フィードバック機能:**
  - スキャン成功時やエラー時（ポイント不足や重複付与など）に、異なる音声トーンとバイブレーションでわかりやすく通知します。

## 🛠 使用技術

- **フロントエンド:** HTML, CSS, Vanilla JavaScript
- **バックエンド (データベース):** Firebase Firestore
- **機能:**
  - `html5-qrcode` を利用したカメラでのQRコード読み取り機能
  - スマートフォンでネイティブアプリのように使えるPWA (Progressive Web App) 対応

## 🚀 セットアップ方法

このアプリを動かすためには、Firebaseのセットアップが必要です。

1. [Firebase Console](https://console.firebase.google.com/) にアクセスし、新しいプロジェクトを作成します。
2. 「Firestore Database」を作成します。
3. Webアプリを追加し、発行された構成情報（APIキーなど）を取得します。
4. リポジトリ内の `firebase-config.js` を開き、取得したご自身のFirebase設定情報に書き換えてください。

\`\`\`javascript
// firebase-config.js の設定例
export const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.firebasestorage.app",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
\`\`\`

## 📱 使い方（スタッフ向け）

1. 本システムをブラウザで開きます（スマホのホーム画面に追加することでPWAとして全画面表示で利用可能です）。
2. 初めての利用者が来た場合は**「👥 利用者一覧・登録」**から任意のIDと名前を入力してユーザーを登録します。
3. 来所時には**「📅 出席登録 (+10pt)」**をタップし、カメラで利用者のQRコード（IDをテキスト化したもの）を読み込みます。
4. 商品を交換する場合は**「🎁 商品交換」**をタップし、希望する商品を選択した上で、利用者のQRコードを読み込みます。
